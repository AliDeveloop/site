        راهنمای دریافت API Key از Google AI Studio

        توجه : api دستیار هوش مصنوعی ایران رو تحریم کرده برای استفاده از DNS یا VPN استفاده کنید

        برای استفاده از این ابزار، نیاز به یک کلید API از Google دارید. مراحل زیر را دنبال کنید:

        1. به سایت Google AI Studio بروید.
        2. اگر حساب گوگل ندارید، یک حساب بسازید یا با حساب موجود وارد شوید.
        3. پس از ورود، روی دکمه "Create API Key" یا "ساخت کلید API" کلیک کنید.
        4. کلید API که برایتان ساخته می‌شود را کپی کنید (یک رشته بلند از حروف و اعداد).
        5. به این صفحه برگردید، کلید را در باکس "کلید API" وارد کنید و "ثبت کلید" را بزنید.
